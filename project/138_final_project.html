III. Analysis

Model Fitting and model selection

Since we have 7 predictor variables, it’s time consuming to do all subset algorithm because there will be \(2^7 = 128\) combination of models. Instead, we implemented backward-forward subset selection and use AIC as a criteria. This combination of model selection can help us find the predictive model. After this, we will do hypothesis testing to check if the selected model fits well.

Models LL DF AIC BIC
Null -210.5825 198 425.1651 431.7617

Models -189.6085 184 411.2169 463.9900

From the table, the AIC and BIC of the selected model is the smallest among three models. However, to check if the selected model fits well, we do the likelihood ratio test to compare the selected model and full model.

\(H_0\): Selected model fits well

\(H_1\): Full model fits well

Test Statistic = \(G^2 = -2(L_0 - L_1) = -2(-191.8765 - (-189.6085))) = 4.536\) with degree of freedom = 6. The corresponding p value is 0.6045.

Selected model

Since p-value > 0.05, we fail to reject \(H_0\) and we can conclude that the selected model fits well. The selected model is the model with variables including gender, schtyp, read, and honors.

Goodness of Fit test

After fitting models, we split the data into two group. First group contains those in middle and high socioeconomic class and second group contains those in low and high socioeconomic class. Then we fit two logistic model seperatly for each group and do model diagnostic for each of them to test if the model fits well and also find out the potential influential points.

For Hosmer and Lemeshow goodness of fit test, the null hypothesis is \(H_0\): the logistic model fits well and the alternative hypothesis if \(H_1\): the logistic model did not fit well. We do the hypothesis test for both logistic regression model. For the first group (Middle and High), the test statistis is 2.2847 with 6 degree of freedom and p-value = 0.8917. For second group (Low and High), the test statistis is 3.1525 with 6 degree of freedom and p-value = 0.7895.
